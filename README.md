# Soultrek India Trekking Analytics

## Project Overview

This project demonstrates SQL analysis using Snowflake on a custom trekking and tourism dataset containing 1,000 booking records.

## Objective

Perform business-driven SQL analysis to answer real-world analytical questions.

## Tools Used

- Snowflake
- SQL
- Excel
- Power BI (Upcoming)

## Dataset

- 1,000 Trek Bookings
- 10 Trek Locations
- Customer Information
- Booking Status
- Revenue
- Discounts
- Payment Methods
